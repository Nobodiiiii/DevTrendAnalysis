
# DevTrendAnalysis

简单的计算机语言、框架、程序员薪资的趋势分析项目。

- **后端**：Python + FastAPI  
- **前端**：React + Vite  

---

## 一、环境要求

- Python 3.10+
- Node.js 18+（自带 npm）
- Git

> 本项目使用 Python 虚拟环境（`.venv`）管理后端依赖，前端依赖通过 `npm` 管理。

## 三、后端环境配置（FastAPI）

### 1. 创建并激活虚拟环境

**Windows（PowerShell）：**

```bash
python -m venv .venv
.\.venv\Scripts\activate
```

**macOS / Linux：**

```bash
python -m venv .venv
source .venv/bin/activate
```

激活成功后，终端前面会出现形如 `(.venv)` 的前缀。

### 2. 安装后端依赖

> `requirements.txt` 在 `backend/` 目录下。

```bash
pip install -r backend/requirements.txt
```

如需使用环境变量（数据库连接、API 密钥等），可在项目根目录创建 `.env` 文件（请勿提交到 Git）。

### 3. 启动后端服务

在项目根目录运行：

```bash
uvicorn backend.app.main:app --reload --host 127.0.0.1 --port 8000
```

启动成功后可访问：

* 接口文档（Swagger）：[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)
* 健康检查（如果实现）：[http://127.0.0.1:8000/health](http://127.0.0.1:8000/health)

## 四、前端环境配置（React + Vite）

### 1. 安装前端依赖

```bash
cd frontend
npm install
```

### 2. 启动前端开发服务器

```bash
npm run dev
```

终端会输出类似信息：

```text
  ➜  Local:   http://127.0.0.1:5173/
```

在浏览器中打开该地址即可访问前端页面。

> 默认前端会通过 `http://127.0.0.1:8000` 调用后端接口，请确保后端已启动。
